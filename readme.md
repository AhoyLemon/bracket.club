# tweetyourbracket.com

## Quickstart

1. Install [`node`](http://nodejs.org/)
2. Install [`npm`](http://npmjs.org/)
3. `git clone git@github.com:lukekarrys/tweetyourbracket.com.git`
4. `cd tweetyourbracket.com`
5. `npm install`
6. `npm start`
